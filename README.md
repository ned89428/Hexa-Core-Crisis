# 能量臨界:六角防線(Hexa-Core Crisis)

1v1 六角網格即時策略塔防 — 玩法驗證 Demo。

**[線上試玩(GitHub Pages)](https://你的帳號.github.io/hexa-core-crisis/)**

## 核心玩法

- **單一資源池**:源能同時供應蓋塔、放地形、召喚怪物 — 每一筆花費都是抉擇
- **賽前佈局**:像下棋一樣用地形版塊(障礙/資源/高台)拚出戰場,免費 8 塊,超額扣開局源能
- **手牌系統(Rogue-lite)**:塔、地板、怪全部抽卡,打出即消耗
- **待命區與羈絆**:怪物先囤進待命區,湊齊羈絆(疾風陣/鐵壁陣/葫蘆小隊/空襲編隊)再整隊出擊
- **擊殺懸賞**:你的怪被擊殺,對手獲得造價 30% 的源能 — 無腦送頭等於資敵
- **動態路徑**:塔即是牆,雙方不斷改建,怪物即時重算最短路;飛行怪無視迷宮
- **反僵局機制**:塔價通膨(越蓋越貴)+ 怪物時間成長(越晚越強),保證對局終結
